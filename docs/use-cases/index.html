4. Use Cases

Use Case Example 1: Scheduling 1-on-1 with a team member

Find a mutually available time with another person, add the event to Google Calendar, and notify via Slack. Then, create a workflow for this process to make it reusable.

Step 1: Register or log in on the product page. (https://studio-pro.wrtn.ai (opens in a new tab))

Step 2: In the prompt field, type a request to schedule a meeting with a specific person (include their email), specify the timeframe for the meeting (e.g., next week between certain days), and ask to add it to Google Calendar.

Step 3: If a prompt appears asking you to connect your account to a specific service to perform the requested actions (e.g., viewing and creating events), simply click the dropdown menu and select the connected account. (If the service connector has not been connected before, an account authentication process will be required.)

Step 4: Check the intermediate results as the AI agent performs actions based on your request, then provide approval or additional comments to proceed.

Step 5: If all the requested actions have been completed, ask to create an automated workflow for the task.

Step 6: If the AI agent creates a workflow, click the ‚ÄòGo to Workflow‚Äô button to navigate to the Workflow Canvas.

Step 7: Click on the nodes of the created workflow to modify their detailed properties in Inspector. (For example, delete the email address entered in the previous conversation and input the email address of the new person to invite.)

Step 8: In the service connector, click the authentication information dropdown menu to connect your account. Then, fill in the empty fields with the appropriate values.

Step 9: Once all the fields are filled in, click Run to execute, and check the results in the execution history.

Use Case Example 2: Image and copy text generation

Create a workflow that repeatedly generates marketing copy and image materials based on the information you input. Build a widget from this workflow to view the automated results on the dashboard.

Step 1: Register or log in on the product page. (https://studio-pro.wrtn.ai (opens in a new tab))

Step 2: In the prompt field, type a request to generate marketing copy and images for your product.

Step 3: Please provide the detailed information requested by the agent to perform the task.

Step 4: When the agent creates the workflow, click the ‚ÄúGo to Workflow‚Äù button to navigate to the Workflow Canvas.

Step 5: Click on the node and enter the values you want to change or add from the pre-filled template.

Step 6: Click the Run button in the upper-right corner and check the generated results in the Execution History tab.

Step 7: To reuse the workflow and change the generated content, click the existing node and update the entered values to the desired content.

Step 8: Check the newly generated results in the Execution History tab, and click Export to register this workflow as a widget.

Step 9: Please select the export format you want.

Step 10: Enter the name and description of the content to be exported, then click the Next button in the upper-right corner.

Step 11: Check the preview of the content to be exported, then click the Next button in the upper-right corner.

Step 12: Please set the conditions for automatically running the workflow repeatedly.

Step 13: You can check the generated widget when you enter the dashboard.

Use Case Example 3: Company report generation for investment

Find detailed information about the company for investment purposes and generate a report.

Step 1: Register or log in on the product page. (https://studio-pro.wrtn.ai (opens in a new tab))

Step 2: In the prompt field, type a request to gather comprehensive investment details and ask for a detailed corporate research report to be written and organized into a document.

Step 3: Please provide the additional information required by the requested service to create the document.

Step 4: Check that the document has been fully completed as requested.

Use Case Example 4: Fetching a Tweet and Summarizing

Gather and summarize tweets from influencers in specific fields like economics or politics, then compile them into a document or share them.

Step 1: Register or log in on the product page. (https://studio-pro.wrtn.ai (opens in a new tab))

Step 2: In the prompt field, type a request to summarize the tweets of people whose tweets you want to gather all at once. Please provide the required account information at this time.

Step 3: Check the quantity and content of the information found by the agent and provide additional instructions as needed.

Step 4: Please input how to save or send the information to use it, and provide any additional information needed for this purpose.

Step 5: Request to create a workflow for detailed editing and repetitive use of this task.

Step 6: Run the workflow to check the results.

Step 7: Please select the export format you want.

Step 8: Enter the name and description of the content to be exported, then click the Next button in the upper-right corner.

Step 9: Check the preview of the content to be exported, then click the Next button in the upper-right corner.

Step 10: Please set the conditions for automatically running the workflow repeatedly.

Step 11: You can check the generated widget when you enter the dashboard.

Use Case Example 5: Reading a GitHub Project and Creating Contributions

Review the GitHub repository, identify areas for improvement or additional features, and suggest ways to contribute to the project.

Step 1: Register or log in on the product page. (https://studio-pro.wrtn.ai (opens in a new tab))

Step 2: In the prompt field, type a request to review a specific GitHub repository and suggest potential ideas for contributing to the project.

Step 3: Please select a specific project from the suggested ones and provide guidance on what to consider during the selection.

Step 4: In the prompt field, type a request to initiate a new development phase by branching off, documenting progress incrementally, and submitting the work for peer review. Include guidance on involving specific team members for feedback and mention any preliminary research or tools that may be required.

Step 5: Complete the implementation and commit the final changes to the branch.

Step 6: Create a pull request for the completed work and submit it.

Step 7: Complete all the steps and provide a link to the pull request.

Step 8: Review the progress of the project through the provided link.

Use Case Example 6: Developing a storytelling agent that enables users to create and export personalized picture books directly to Google Slides.

Step 1: Register or log in on the product page. (https://studio-pro.wrtn.ai (opens in a new tab))

Step 2: In the prompt field, type a request to create an AI agent capable of generating a personalized story and picture book.

Step 3: You will be prompted to connect to your Google Slides account. Select your account and log in.

Step 4: The Studio autonomous agent will then create a workflow to develop the storyteller bot and generate a unique picture book. Click the ‚ÄúView Workflow‚Äù button on the workflow card to see the details.

Step 5: View the workflow in the Canvas. Please fill in the guidance for the story agent‚Äôs responses in the right inspector panel.

Step 6: Click the ‚ÄúRun Chatbot‚Äù button in the top right corner.

Step 7: The storybook author bot will be launched. You can provide instructions on the theme or characters for the story. Then, it will generate a personalized picture book based on your request. (Currently, story creation is only supported in Korean.)

Step 8: Please connect the required account information to generate the completed story in Google Slides.
