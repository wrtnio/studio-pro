Meta LLM 채팅 세션 메시지의 데이터 정보.

IStudioMetaChatSessionMessageData 는 IStudioMetaChatSession Meta LLM 채팅 세션에서 발생한 IStudioMetaChatSessionMessage 메시지의 상세 데이터를 기록한 엔티티이다.

그리고 우리의 Meta LLM 채팅 세션은 웹소켓 프로토콜에 TGrid 의 RPC (Remote Procedure Call) 개념을 사용하고 있기에, 본 IStudioMetaChatSessionMessageData 는 곧 RPC 함수 호출에 대한 이력 정보를 뜻하는 엔티티이기도 하다.
