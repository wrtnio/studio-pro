Interface IStudioRepositoryReleaseNode

Information about individual nodes that make up the distribution program of the repository.

IStudioRepositoryReleaseNode is an entity that embodies programming nodes that the user has specified and exported within the program distributed from the repository.

For reference, the types of nodes that can be exported include user-defined functions and workflow documents.

Of course, IStudioWorkflow workflow document also results in a single function as a build result, so ultimately, a node means a "reusable function" exported from this repository.

In addition, each exported node has its own endpoint on the Rest API. Therefore, each node in the repository can be used by importing in other programs, but it is also possible to call it externally through the Rest API. And it is also possible to register and sell it as a IHubSale listing of Generative Hub.
