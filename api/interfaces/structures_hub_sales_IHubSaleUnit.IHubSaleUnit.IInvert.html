Interface IInvert

Reverse reference information of the sale unit.

IHubSaleUnit.IInvert is a data structure interface that visualizes the reverse reference information of the sale unit from the perspective of IHubCartCommodity, and therefore, in terms of DB concept, it is a structure corresponding to IHubCartCommodityStock.

Therefore, IHubSaleUnit.IInvert does not have all IHubSaleUnitStock records of the sale unit, but only has elements that are in the IHubCartCommodity shopping cart.
