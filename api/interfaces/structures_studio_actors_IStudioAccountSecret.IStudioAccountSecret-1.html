Interface IStudioAccountSecret

스튜디오 계정에 등록된 시크릿 변수 세트 정보.

IStudioAccountSecret 은 IStudioAccount 스튜디오 계정에 등록된 시크릿 변수의 세트를 형상화한 엔티티이다. 하나의 시크릿 key 에 대하여, 복수의 변수값들을 모아둔 개념이다. 그리고 여기서 말하는 시크릿 변수란, 타 어플리케이션과의 연동에 사용하는 인증 키나 비밀번호 등을 의미한다.

참고로 시크릿 변수의 주 목적은, IStudioWorkflow 워크플로우 프로그램 내에서, 커넥터나 워크플로우 함수에 삽입하는 파라미터 또는 일부 속성으로써 사용하기 위함이다. IAstWorkflowSecretIdentifier 를 통하여 식별하며 주로 구글이나 페이스북 등의 인증 키를 저장하는데 사용된다.

그리고 이러한 목적 때문에, 변수값은 DB 에 암호화하여 저장한다.
