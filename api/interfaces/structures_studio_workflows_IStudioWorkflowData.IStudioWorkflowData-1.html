Interface IStudioWorkflowData

The main data of the workflow document.

IStudioWorkflowMetadata is an entity that visualizes the main data of the workflow document.

It contains information about each connector node and their connection codes placed in the workflow document. And since the workflow document is also interpreted as a function, meta information about the workflow itself is also included.

For reference, the reason connector is separated from each node is because one connector function can be used in multiple connector nodes (IStudioWorkflowCallStatement). And the reason why each node of each workflow is not a tree structure like an AST node in programming, but a key-mapping structure that indirectly references IStudioWorkflowStatement.id is also similar. This is because each node of the workflow can establish a circular recursive relationship depending on the arrow connection relationship between them.
