Interface IStudioWorkflowCallStatement

A connector function call statement in the workflow.

IStudioWorkflowCallStatement is a node entity that visualizes an expression statement that calls the IStudioWorkflowConnector connector function in the IStudioWorkflow workflow program.

When configuring IStudioWorkflowCallStatement, specify the target connector you want to call with connector, and enter the list of parameters to be assigned when calling the connector in arguments as a AST expression.

As a reference, whenever this workflow system calls a connector function, it stores the return value in a some local variable. And this some local variable is identified by IAstWorkflowCallStatementResultIdentifier. However, if you explicitly specify operand, it should be stored in IStudioWorkflowVariable and identified by IAstWorkflowVariableIdentifier.

In addition, for title and descrition fields, read the supertype IStudioWorkflowNodeDescriptable.

const result = connector(...arguments);

nexts

nexts: IStudioWorkflowNodeNext[]

List information of the next node statement.

After the execution of the current node statement is completed, a list of IStudioWorkflowStatement.id and a IStudioWorkflowNodeNext structure that describes the node statements to be executed next.

Note that the node that can be referred to must belong to a block at the same level as the current node. If the current node refers to a node belonging to a block such as IStudioWorkflowForEachStatement for loop or IStudioWorkflowBranchStatement if statement, or a node belonging to a block such as a for loop or if statement refers to a node outside the block it belongs to, a validation error occurs by the workflow compiler.

Optionaloperand

operand?: null | IAstWorkflowVariableIdentifier

A variable to store the connector call result.

If this value is not set, the connector call result is identified only by IAstWorkflowCallStatementResultIdentifier.

Conversely, if this value is set, the connector call result is stored in a user-defined variable IStudioWorkflowVariable, which must be identified by IAstWorkflowVariableIdentifier.
