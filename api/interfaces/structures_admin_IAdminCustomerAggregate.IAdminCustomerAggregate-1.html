Interface IAdminCustomerAggregate

Customer information managed by the administrator.

IAdminCustomerAggregate is an entity that visualizes customer information managed by the administrator.

You can use IHubMember.created_at to express the member conversion date.

  • If IHubMember.seller is not null, it means that it is a Seller member.
  • If IHubMemberVillain is not null, it means that it is a Villain member.
  • If IHubMemberElite is not null, it means that it is an Elite member.
