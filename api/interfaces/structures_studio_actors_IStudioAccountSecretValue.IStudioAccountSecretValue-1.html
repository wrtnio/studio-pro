Interface IStudioAccountSecretValue

Global variable value information registered in the studio account.

IStudioAccountSecretValue is an entity that embodies the individual values of the secret variable set registered in the IStudioAccount studio account.

And the secret variable referred to here refers to authentication keys or passwords used for linking with other applications. For reference, the main purpose of the secret variable is to be used as a parameter or some property inserted into a connector or workflow function within the IStudioWorkflow workflow program.

It is identified through IAstWorkflowSecretIdentifier and is mainly used to store authentication keys such as Google or Facebook.

And for this purpose, variable value is encrypted and stored in the DB.
