Interface IAstArrayFunctionIndexIdentifier

Index number identifier in Array.map() function.

ArrayFunctionIndexIdentifier is an interface used to identify the index number of the callback function in the array conversion function expression.

For example, if the array conversion function expression used is IAstArrayMapExpression, then ArrayFunctionIndexIdentifier is used to identify index in Array.map((_, index) => ({ ... })) and the value of IAstArrayMapExpression.id must match IAstArrayFunctionIndexIdentifier.expression_id.

Below is an example of using IAstArrayFunctionIndexIdentifier and IAstArrayMapExpression.

Code Example

const array: number[] = [...];
array.map((value) => {
  return {
    id: value + index,
    age: 20,
  };
});
