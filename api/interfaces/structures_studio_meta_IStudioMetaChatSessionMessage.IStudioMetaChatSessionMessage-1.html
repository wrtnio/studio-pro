Interface IStudioMetaChatSessionMessage<Data>

Meta LLM chat session message information.

IStudioMetaChatSessionMessage is an entity that records messages that occurred in a IStudioMetaChatSession Meta LLM chat session. To be precise, it records which IStudioMetaChatSessionConnection WebSocket connection within the chat session it occurred in.

And since our Meta LLM chat session uses TGrid's RPC (Remote Procedure Call) concept in the WebSocket protocol, this IStudioMetaChatSessionMessage is also an entity that encompasses IStudioMetaChatSessionMessageData, the history information for RPC function calls.

speaker

speaker: "customer" | "agent"

Who is the speaker?

It means who is the originator of the current message, and conversely, it also means the type of RPC function caller.

That is, if this attribute value is "customer", the client called the server's IStudioMetaChatService RPC function, and conversely, if this attribute value is "agent", the server called the client's IStudioMetaChatListener RPC function.

  • customer: Customer (human)
  • agent: LLM Agent (robot)
