Interface IStudioRepositoryFile

File information of the repository.

IStudioRepositoryFile is an entity that visualizes files existing in IStudioRepository repository, and is a type of IStudioRepositoryBucket bucket.

And when a file is first created, and whenever it is modified or its path is changed, its history is stored in the DB as a unit called a snapshot and tracked. In addition, all modifications to these files are grouped into a unit called IStudioRepositoryCommit commit and are done at once. Therefore, by viewing past files, it is also possible to view the previous history of each file.
