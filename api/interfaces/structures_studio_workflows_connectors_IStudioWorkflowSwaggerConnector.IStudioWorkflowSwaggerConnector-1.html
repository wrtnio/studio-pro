Interface IStudioWorkflowSwaggerConnector

Connector information of Swagger function in workflow.

IStudioWorkflowSwaggerConnector is an entity that visualizes the connector of Swagger function placed in IStudioWorkflow workflow. And the "Swagger function" here means an external Rest API and its metadata spec that can be called with the IStudioWorkflowCallExpression node in the workflow document.

If the function you want to call is not from an external Rest API server but is written in TypeScript in the same IStudioRepository repository, use the IStudioWorkflowScriptConnector type instead. And if the function you want to call is exported (export) from the IStudioRepositoryRelease release of another repository, use the IStudioWorkflowReleasedConnector type instead.
