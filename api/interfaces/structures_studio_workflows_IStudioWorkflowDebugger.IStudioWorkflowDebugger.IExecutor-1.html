execute

  • execute(props): Promise<IStudioWorkflowDebugger.IResult<any>>

  • Executes a workflow.

    Executes the target IStudioWorkflow workflow function in debugging mode, and receives the progress of the workflow through the IListener that has already been registered.

    Note that the execute function can be re-executed repeatedly. However, this cannot be done while the previously executed workflow is still in progress, and in this case, an error occurs. Therefore, if there is a previously executed workflow, wait until it ends, or forcefully stop it using the terminate function and re-execute it.
