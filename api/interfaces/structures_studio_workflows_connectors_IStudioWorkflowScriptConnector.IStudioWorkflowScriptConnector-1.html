Interface IStudioWorkflowScriptConnector

Script function connector information in the workflow.

IStudioWorkflowScriptConnector is an entity that visualizes the connector of the script function placed in IStudioWorkflow workflow. And the "script function" here means a TypeScript function written in a IStudioRepositoryFile file somewhere in the IStudioRepository repository that can be called with a IStudioWorkflowCallExpression node in the workflow document.

If the function you want to call is not written in TypeScript in the same repository, but exists in the IStudioWorkflowRelease release of an external repository,

use the IStudioWorkflowScriptConnector type instead. And if the function you want to call is an external Rest API, use the IStudioWorkflowSwaggerConnector type instead.
