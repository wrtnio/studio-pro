Interface IStudioShelfReleaseWorkflow

릴리즈된 워크플로우 프로그램 책장에 꽂힘.

IStudioShelfReleaseWorkflow 는 고객이 "배포장의 책장에 꽂은 워크플로우 문서" 를 형상화한 엔티티이다.

참고로 "배포장의 책장에 꽂혀있다" 라는 개념은, 해당 워크플로우 문서가, 최소 한 번 이상 릴리즈되었음을 뜻한다. 만약 해당 워크플로우 문서가 아직 배포되지 않았다면, 이 때는 작업장의 책장을 뜻하는 IStudioShelfCommitWorkflow 로 분류된다.

그리고 본 엔티티 상 워크플로우의 접근 경로는 (IStudioAccount 계정 -> 저장소 -> 릴리즈 -> 워크플로우) 가 아닌, 책장에서 모두 한 번에 모아보는 개념이다. 따라서 본 엔티티에는 귀속 저장소와 릴리즈 정보가 함께 들어있다. 그리고 오직 가장 최신 릴리즈의 워크플로우만이, 책장에 꽂힌다.
