Interface IHubSaleAuditApproval

Approval for listing review.

IHubSaleAuditApproval is an entity that embodies the action of administrator approving IHubSaleAudit listing review.

Note that the administrator who initiated the IHubSaleAudit review and the administrator who processes the approval can be different people. Also, it is possible to IHubSaleAuditRejection reject the review and then reverse and approve it. However, it is impossible to reverse and reject an already approved review.

This is because the listing is actually sold upon approval.
