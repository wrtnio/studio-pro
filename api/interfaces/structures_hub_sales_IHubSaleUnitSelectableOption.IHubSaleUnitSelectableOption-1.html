Interface IHubSaleUnitSelectableOption

Information on individual selectable options in the listing unit.

IHubSaleUnitSelectableOption is a sub-entity of IHubSaleUnit that visualizes individual products in the listing, and is an entity designed to visualize individual selectable option information of the unit.

  • Examples of selectable options
  • Computer performance: CPU, GPU, RAM, etc.
  • Image generation AI: Rendering quality, license type, theme, etc.

And if the attribute value IHubSaleUnitSelectableOption.variable of this entity is true, this means that the IHubSaleUnitStock final stock to be purchased will vary depending on which candidate value the customer selects in this option.

Conversely, if the above value is false, then whatever candidate value the customer selects from the options seen will not affect the IHubSaleUnitStock final stock that the customer will purchase. It is just simple information that the seller will use for reference only.
