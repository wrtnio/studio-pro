Interface IStudioWorkflow

Workflow document file.

IStudioWorkflow is an entity that embodies a workflow document file, and is a type of file managed by repository, so it is also a subtype of bucket.

Therefore, whenever a workflow document is modified, a new snapshot record is issued and accumulated, and this editing is done in units of commit.

In addition, the studio's workflow document embodies components for visual programming. Therefore, the workflow document can actually be converted into programming code (TypeScript), and becomes a standalone program when built (compiled + bundled).

Of course, you can release the associated repository to build and distribute other workflow documents or connector functions together with this workflow document.

  • Workflow-only build: For testing
  • Release: For distribution
