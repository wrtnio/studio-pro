Interface IForSale

Information about the distribution program from the repository.

IStudioRepositoryRelease is an entity that embodies a program distributed to the IHubRepository repository targeting the IHubRepositoryCommit commit feature.

Therefore, the IStudioRepositoryRelease entity consists of IStudioRepositoryReleaseNode records based on the elements (functions) selected by hub_customers customers from each of the IStudioRepositoryFile file and IStudioWorkflow workflow buckets that make up the repository. And the compiled and bundled results are stored along with metadata (type) information about them.

Distributing them widely is a release.
