Outline

"Wrtn Studio Pro" is a service providing an evolved Meta LLM (A.I. chatbot) than ever.

"Wrtn Studio Pro" provides a Meta LLM (A.I. Chatbot) which can perform various function callings obtained from API marketplace sales of OpenAPI specification. "Wrtn Studio Pro" also provides an Workflow Compiler (+SWL language) which automates the Meta LLM scenario to an re-usable and executable program function with visual diagrams.

In here technical document, you can learn how the "Wrtn Studio Pro" has developed the Meta LLM (A.I. chatbot) and how it obtains the function calling schemas from the API marketplace. Also, you can learn how the user utternance and LLM function call histories are reborn as reusable programs through the workflow compiler.
