name

name: IAstIdentifier | IAstWorkflowCallStatementResultIdentifier | IAstWorkflowLoopStatementResultIdentifier

속성 식별자 표현식.

만약 사용되는 name 객체의 타입이 IAstIdentifier 가 아닌 IAstWorkflowCallStatementResultIdentifier 나 IAstWorkflowLoopStatementResultIdentifier 라면, 이는 expression 아래의 상황에 해당한다.

즉 expression 을 통하여 간접 참조하는 객체가, IStudioWorkflowForEachStatement 나 IStudioWorkflowWhileStatement 등의 반복문을 통하여 생성된 배열 변수인 것.

{
  type: "PropertyAccessExpression",
  expression: {
    type: "ElementAccessExpression",
    expression: {
      type: "WorkflowLoopStatementResultIdentifier",
      loop_statement_id: "some_for_each_statement_id",
    },
    questionDotToken: null,
    argumentExpression: {
      type: "NumericLiteral",
      text: "0",
    },
  },
  questionDotToken: null,
  name: {
    type: "WorkflowCallStatementResultIdentifier",
    call_statement_id: "some_call_statement_id_placed_in_a_block_of_loop_node",
  },
}
