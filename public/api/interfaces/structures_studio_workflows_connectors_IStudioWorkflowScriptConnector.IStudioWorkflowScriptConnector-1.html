Interface IStudioWorkflowScriptConnector

워크플로우상 스크립트 함수의 커넥터 정보.

IStudioWorkflowScriptConnector 는 IStudioWorkflow 워크플로우에 배치한 스크립트 함수의 커넥터를 형상화한 엔티티이다. 그리고 여기서 말하는 "스크립트 함수" 란, 워크플로우 문서에서 IStudioWorkflowCallExpression 노드로 호출할 수 있는 동 IStudioRepository 저장소의 어딘가 IStudioRepositoryFile 파일에 쓰인 타입스크립트 함수를 뜻한다.

만일 호출하고자 하는 함수가 동 저장소에 타입스크립트로 작성된 것이 아닌, 외부 저장소의 IStudioWorkflowRelease 릴리즈 본에 존재한다면 IStudioWorkflowScriptConnector 타입을 대신 사용하도록 한다. 그리고 만일 호출하고자 하는 함수가 외부 Rest API 라면, IStudioWorkflowSwaggerConnector 타입을 대신 사용한다.
