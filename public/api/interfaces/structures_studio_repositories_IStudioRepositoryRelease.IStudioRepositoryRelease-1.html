Interface IStudioRepositoryRelease

저장소로부터의 배포 프로그램 정보.

IStudioRepositoryRelease 는 IHubRepository 저장소에서 특징 IHubRepositoryCommit 커밋을 대상으로 배포한 프로그램을 형상화한 엔티티이다.

따라서 본 IStudioRepositoryRelease 엔티티에는 저장소를 구성하는 각 IStudioRepositoryFile 파일 및 IStudioWorkflow 워크플로우 버킷들로부터, hub_customers 고객이 선택한 요소 (함수) 들을 기점으로하여 IStudioRepositoryReleaseNode 레코드들이 구성된다. 그리고 이들에 대한 메타데이터 (타입) 정보와 함께, 컴파일 및 번들링된 결과물이 저장된다.

이들을 널리 배포하는 것이 바로 릴리즈인 것.
