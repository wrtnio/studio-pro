Function bindable

워크플로우 데이터에서 치환 가능한 대상들을 추출함.

워크플로우 데이터에서 사용된 모든 커넥터 함수 호출문을 순회하며, 그곳에 입력한 인자값들 IStudioWorkflowCallStatement.arguments 를 순회하며, 각각 파라미터로 치환 가능한 대상을 추출해낸다. 참고로 치환 가능 argument 란 곧, literal expression 이다.

그리고 리턴값 중 일부 원소들에 대하여 IBindable.replace 함수를 호출하면, 그 즉시로 IStudioWorkflowData.arguments 에 치환 대상들이 추가되고, IStudioWorkflowData.parameters 에도 치환 대상들의 정보가 추가되며, IStudioWorkflowCallStatement.arguments 에는 치환 대상들이 대체된다.
